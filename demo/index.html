Demos as Guests

All those demos connect as Guest to the API test account. The media stream is handled by default by the API. It supposes 1 audio and 1 video input, and ask for a VGA resolution (640*480). More advanced stream manipulation can also be hooked up by passing a stream table to the API through the user (Demo to come). That allows audio only calls for example, or multiple input calls or resolution changes, .....

Demo 1

A simple Demo. Good candidate for a widget.

  • Automatically fetch for cam.
  • Automatically connects to default Room.
  • Handle connection and disconnection of peers.
  • Can connect and interop with www.webrtc-enterprise.com/room/apitest.
                // ROOM_SERVER    - The room server URL where SkywayJS fetches from
                // APP_ID         - The registered Application ID
                // ROOM           - The room name
                // AUTO_FETCH_CAM - Flag to automatically fetch camera after Server information are parsed. Enable this if Call requires video to start chat.

                var skyway = new Skyway();
                skyway.init(ROOM_SERVER, APP_ID, ROOM, AUTO_FETCH_CAM);

Demo 3

An additional add-on implementation of Demo 2, where users can set their own start date, time and duration of the call.

  • User has to generate credentials by their own. Hash and credentials is generated using CryptoJS library in this demo.
  • User is able to set the duration and the start date and time of the call
                // ROOM_SERVER     - The room server URL where SkywayJS fetches from
                // APP_ID          - The registered Application ID
                // ROOM            - The room name
                // AUTO_FETCH_CAM  - Flag to automatically fetch camera after Server information are parsed. Enable this if Call requires video to start chat.
                // SECRET          - The secret token of the Application provided
                // START_DATE_TIME - The start date and time of the call in Date ISO String format. 
                // DURATION        - The duration of the call.
                // CREDENTIAL      - The generated credential of the secret provided using the start date and time, room and duration.

                var hash = CryptoJS.HmacSHA1(ROOM + '_' + DURATION + '_' + START_DATE_TIME, SECRET);
                CREDENTIAL = encodeURIComponent(hash.toString(CryptoJS.enc.Base64));
                var skyway = new Skyway();
                skyway.init(ROOM_SERVER, APP_ID, ROOM, AUTO_FETCH_CAM, START_DATE_TIME, DURATION, CREDENTIAL);
