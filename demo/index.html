Demos as Guests

All those demos connect as Guest to the API test account. The media stream is handled by default by the API. It supposes 1 audio and 1 video input, and ask for a VGA resolution (640*480). More advanced stream manipulation can also be hooked up by passing a stream table to the API through the user (Demo to come). That allows audio only calls for example, or multiple input calls or resolution changes, .....

Demo 2

A moderately complete demo. Note that you can join a room without having a local stream, in which case you can see other people, but they can;t see you. Chat and other functionalities are of course available once in the room.

  • With list of users in room.
  • Feedback on channel status.
  • Feedback on handshake and Connection.
  • Getting video and joining room separated.
  • File sharing features using DataChannel.
